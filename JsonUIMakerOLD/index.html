About The Project

The world is increasingly being taken over by AI. In the context of today's AI's, programming is sometimes possible without any programming knowledge. To implement this even further, so-called “NoCode” platforms are often offered for “programming” in conjunction with a graphical editor. This concept fascinates me and JsonUI-Maker is my contribution to this incredibly useful environment. I originally developed the concept of this editor so that I would no longer have to do boring copy/paste for my own forms. However, during the development, which only lasted a week, I realized that I wanted to make this tool available to every Minecraft enthusiast.
